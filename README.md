# ソ・ラキメ共和国
(Wikipedia風に行きますね)

ソ・ラキメ共和国（そ・らきめきょうわこく、英:Republic of So Rakime）は、異世界にある共和制国家。首都はソ・ラ県首都市。
この国が存在する世界のこの国では、南北の気候の差が大きく、四季は見られないが真夏のように熱い地域のほか極寒の地も存在する。
